# Microstream

Microstream is a lightweight self hosted microblog app written in PHP that runs on shared hosting and lets sites follow each other using the Microstream Protocol v1.0.

## Features

- Short form microblog posting
- Simple, shared hosting friendly PHP stack
- JSON over HTTPS federation using the Microstream Protocol
- Pull based remote timelines, optional push events

- PHP 8+
- MySQL or MariaDB
- Apache or Nginx with HTTPS
